Interaction principles

Only use status indicators:

  • to tell the user something important
  • to inform the user of a change
  • to inform the user that they need to do something

For less prominent status indicators you could use labels.

Content principles

Clear content about the change and what it means for the user.

Services using statuses

  • Returns and movement
  • Passport renewals
  • Passport application search
